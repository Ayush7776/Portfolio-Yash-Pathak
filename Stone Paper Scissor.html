Stone Paper Scissor Game

Based On - Java

Tool - J.D.K. 17 LTS

The game is two-person game called "Guess the Number". The first player thinks of an integer within a known range. The second player tries to guess the number. If the guess is incorrect, then the first player tells the second player whether the guess was too high or too low.

  • Random.nextInt() function to take random input from computer (player 1) and prompt user (player 2) to choose an option rock, paper, or scissors and finishes the code by adding nested if-else statements to appropriately report “User won”, or “Computer won”, or "No Winner won ( Both choose same )."
